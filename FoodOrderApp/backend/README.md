# Saffron Kitchen - Backend Services

This directory is currently a placeholder for the application's backend architecture.

## 🚧 Project Status

The current implementation of Saffron Kitchen utilizes a **Serverless Architecture** via **Firebase (Firestore)** for all database operations, real-time menu management, and order processing.

As per the current submission requirements, a dedicated custom backend server is out of scope. All business logic and data persistence are handled directly by the frontend's integration with Firebase services.

## 🔮 Future Roadmap

In future iterations, this directory will house:
- **Node.js/Express API**: To handle complex business logic and secure server-side operations.
- **Payment Gateway Integration**: Webhook handlers for Stripe/Razorpay.
- **Advanced Auth**: Custom authentication flows and session management.
- **Push Notification Service**: Centralized handling for order status updates.
